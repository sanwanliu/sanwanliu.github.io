Java底层:GC

标记算法

引用计数算法

  1. 通过判断对象的引用数量来决定对象是否可以被回收
  2. 每个对象实例都有一个引用计数器,被引用则+1,完成引用则-1
  3. 任何引用计数为0的对象实例可以被当做垃圾收集

优点: 执行效率高,程序执行受影响较小

缺点: 无法检测出循环引用的情况,导致内存泄露
