21.递归和分治:leetcode-50

  2019-09-05   2019-09-17
  leetcode / 算法
  a minute read (About 129 words) 0 visits

Implement pow(x, n).

eg. 2^2 = 2^1 * 2^1 = (2^0 * 2^0 * 2) * (2^0 * 2^0 * 2) = (1 * 1 * 2) * (1 * 1 * 2) = 4

eg. 2^3 = 2^1 * 2^1 * 2 = (2^0 * 2^0 * 2) * (2^0 * 2^0 * 2) * 2 = (1 * 1 * 2) * (1 * 1 * 2) * 2 = 8

注意：负数的int的取值范围为-2^31~2^31-1,如果直接使用-n会导致栈溢出

approach1

time: O(logn)
space: O(logn)

approach2

time: O(logn)
space: O(1)
